# Установка и настройка Qt и Qt Creator

## Загрузка и установка

Пакеты для создания OpenSource (лицензии GPL, LGPL) программ можно скачать здесь:
https://www.qt.io/download-open-source/#section-2

Советую взять Offline версию.
Так же советую ставить отдельно сам Qt и отдельно Qt Creator.
Это добавит немного сложности, но позволит обновлять каждый компонент по отдельности.

Пакет выбирается под архитектуру процессора (х32, х64) и под ОС.
Можно поставить одновременно несколько пакетов.

Пакеты с подписью VS 2015, будут использовать компилятор Visual Studio 2015.
С подписью MinGW - Open Source компилятор MinGW (который является портом GCC и G++ под ОС Windows).

Лучше устанавливать в папку, путь к которой не содержит пробелов,
а лучше содержит только латинские символы, подчёркивание и тире. 

При установке, говорят, можно не регистрироваться, а должна быть кнопка "пропустить".

Если устанавливать версию для MinGW, стоит убедиться, что в диалоге установки в меню
Tools выбран пункт MinGW. (Хотя в случае чего его можно скачать отдельно
с официального же сайта Qt http://download.qt.io/development_releases/prebuilt/mingw_32/ )

Так же посмотреть информацию о других доступных для установки компонентов.

## Настройка

Если Qt ставился одним пакетом, вместе с Qt Creator, Qt Creator скорее всего настроен.

Иначе стоит зайти в настройки Qt Creator:
В верхнем меню "Инструменты" - "Параметры".

В открывшемся окне выбрать "Сборка и запуск".
Перейти на вкладку "Компиляторы":
Если там нет нужного компилятора:
Нажать "Добавить" и заполнить появившиеся внизу поля:
название, которое вы хотите дать, и путь к компилятору.
(Проделать для С и С++.)
Для MinGW они могут быть примерно такими:
Для С:
%Папка куда установлен Qt%\Tools\mingw530_32\bin\gcc.exe
Для С++:
%Папка куда установлен Qt%\Tools\mingw530_32\bin\g++.exe

Затем перейти на вкладку "Профили":
Нажать "Добавить" и указать путь к qmake.exe
Он может быть примерно такой:
%Папка куда установлен Qt%\5.8\mingw53_32\bin\qmake.exe

Название: Qt %{Qt:Version} %то, что вы добавите для понимания%

Теперь на вкладку "Комплекты":
Нажать "Добавить".
Задать название комплекта, выбрать компиляторы С и С++, Профиль Qt.
А также дополнительные параметры, если они требуются (например "тип устройтва").

С помощью одного из комплектов и будет собираться программа.

Убедиться, что на вкладке "Текстовый редактор",
"Поведение" установлена кодировка файлов "UTF-8".

Возможно стоит установить размер шрифта и цветовую схему (на чёрную).

## Компиляция и запуск

Создаётся новый проект или открываете существующий.
Если вы не знаете, какой вам создавать - выбирайте "Приложение Qt Widgets".

На левой панеле Qt Creator:
С помощью значка похожего на монитор можно выбрать
один из установленных для проекта комплектов и тип: отладочная версия (debug) или релиз.

Нажав на зелёный треугольник можно скомпилировать и запустить проект.
(Для этого же можно использовать верхнее меню, пункт "Сборка".)

## Для запуска приложения вне Qt Creator под Windows

Потребуется скопировать некоторые DLL:
- для MinGW из папки, куда он был установлен
- для самого Qt:
%Папка куда установлен Qt%\5.8\mingw53_32\bin
И из
%Папка куда установлен Qt%\5.8\mingw53_32\plugins
Минимум папки: iconengines,
imageformats, platforms.
И из
%Папка куда установлен Qt%\5.8\mingw53_32\
скопировать translations.

Ещё можно почитать об этом:
https://wiki.qt.io/Deploy_an_Application_on_Windows

Раздел Creating the Application Package из
http://doc.qt.io/qt-5/windows-deployment.html
