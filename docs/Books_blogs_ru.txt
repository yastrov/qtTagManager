# Книги и блоги

## Некоторые книги по С++:
-  Бьярн Страуструп
Может быть тяжёл для понимания, так что возможно для продвинутого уровня. Чем новее издание, тем лучше.

-  Стивен Прата "Язык программирования C++. Лекции и упражнения" (издание 6 или более новое)
-  Stephen Prata "C++ Primer Plus"
Общий учебник по С++. Чем новее издание, тем лучше (больше шансов, что описывает C++11, C++14, C++17).

-  Скотт Мейерс "Эффективный и современный С++. 42 рекомендации по использованию C++11 и C++14"
-  Scott Meyers "Effective Modern C++", O’Reilly
Книга знаменитого специалиста о новых стандартах и практиках С++. Так же предыдущие книги Мейерса можно почитать для кругозора.

-  Уильямс, Энтони "Параллельное программирование на C++ в действии"
-  Williams, Anthony "C++ Concurrency in Action"
Книга по параллельному и многопоточному программированию на чистом С++11. (Добавлено скорее для полноты и кругозора.)

-  Peter Van Weert Marc Gregoire «C++ Standard Library Quick Reference» 2016
-  Kurt Guntheroth «Optimized C++» 2016, O’Reilly
Книга по приёмам оптимизации программ.

## Книги по Qt и QML:

-  Mark Summerfield "C++ GUI Programming with Qt 4" (2nd Edition)
Обзорная книга по возможностям Qt 4.

-  Марк Саммерфилд - "Qt Профессиональное программирование" - 2011
-  Mark Summerfield - "Advanced Qt Programming: Creating Great Software with C++ and Qt 4"
Не смотря на то, что книга описывает старый Qt 4 и старый С++, большинство материалов применимо и в Qt5.
(Самое заметное на глаз - в Qt 5 появился более удобный синтаксис сигнал-слотов.)
Прочитав её я пожалел, что потратил время на поиск статей по моделям, автодополнению, многопоточности, выводу на принтер... Достаточный аргумент для прочтения? По моему да.
Но автор этим не ограничивается, давая краткий обзор Qt State Machines, которые позволяют сделать проще работу с состояниями приложения (например "Пауза", "Воспроизведение" для проигрывателей.)

-  http://qmlbook.github.io - A book about Qt5 QML (Книга о QML - новой технологии, приходящей на смену виджетам)
-  Symeon Huang "Qt 5 Blueprints" (Частично о Qt, частично QML )
